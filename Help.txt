命令行执行网页：

CMD Mode like this : 

normal模式
php -q index.php controller=index/action=home/id=100
php -q index.php controller=plugins/action=index/page=2

middle模式
php -q index.php controller/index/action/home/id/100
php -q index.php controller/plugins/action/index/page/2

short模式
php -q index.php index/home/100
php -q index.php plugins/page/2.html

PHP要求5.3+


新增功能：

1.支持多级目录
new auto_text_module() => require('module/text/auto.text.module.php'); new auto_text_module();

2.新增smarty的appendParam插件

3.新增smarty的bulidURL插件 Qii::load("Router")->url(array('controller' => 'test', 'action' => 'index', 'page' => 1)) ...;

4.修改原来setError中关闭错误调试及设置错误页面后的跳转为不跳转并直接输出指定错误页面。

5.Controller中的Qii特殊说明，此处的Qii跟instance类似，不过主要用于框架中的类实例化，由于部分类实例化之前需要引用其他的文件，故写在Controller中的QiiClass方法，目前支持view、model、uri、language。

6.增加Controller中instance和__get方法.如$this->instance('module');$this->module->getRow("SELECT * FROM user");

7.增加Controller中dispatch方法，用于程序中再次分发,分发同时还是受路由限制，路由的优先级最高。

8.注册smarty插件
	$this->Qii('View', array('smarty_view_class_plugin' => 'global_helper'));
